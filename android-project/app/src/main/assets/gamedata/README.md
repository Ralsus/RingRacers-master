# Размещение файлов игры

Поместите файлы .pk3 сюда, и они будут включены в APK.

## Необходимые файлы

Скачайте Ring Racers и скопируйте сюда:

- `bios.pk3`
- `chars.pk3` 
- `follower.pk3`
- `gfx.pk3`
- `maps.pk3`
- `music.pk3`
- `patch.pk3`
- `sounds.pk3`
- `textures.pk3`

## Как получить файлы

1. Скачайте Ring Racers с https://www.kartkrew.org/
2. Установите игру на PC
3. Скопируйте все .pk3 файлы из папки игры сюда

## Размер

Общий размер файлов ~2-3 ГБ. APK будет большим!
